## 常用 cmd 命令

| win+R                           | 启动 CMD                         |
| ---------------------- | ---------------------------- |
| cd C: \User\DevProject\ | 切换路径至指定路径           |
| mkdir XXX              | 新建 XXX 文件夹                |
| pwd                    | 查看文件路径                 |
| 磁盘名称:                       | 切换磁盘                        |
| Cd 文件名部分名称+tab           | 切换文件夹内选择的文件/填充文字 |
| Cd 文件夹名称、文件夹名称、文件名 | 直接进入的命令 cd                |
| Cd ..                           | 返回上一级                      |
| Cd \                            | 直接返回根路径磁盘              |
| dir                             | 显示文件夹内容 ( 文件名称预览 ) |
| cls                             | cmd 窗口清屏                     |
| exit                            | 退出                            |
| ↑↓键                            | 快速调换使用过的命令            |
