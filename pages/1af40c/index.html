# 1、

证明x5−3x−1=0x^5 - 3x - 1 = 0x5−3x−1=0 在区间 (1,2)(1, 2)(1,2) 内至少有一个实根

思路：我们可以使用介值定理（Intermediate Value Theorem）来进行证明。

首先，我们需要证明方程在区间上连续。我们知道，多项式函数是连续的，因此 f(x)=x5−3x−1f(x) = x^5 - 3x - 1f(x)=x5−3x−1 在整个实数域上都是连续的。

接下来，我们需要证明在区间的两个端点上函数值的符号不同。计算 f(1)f(1)f(1) 和 f(2)f(2)f(2) 的值：

f(1)=15−3(1)−1=−3f(1) = 1^5 - 3(1) - 1 = -3f(1)=15−3(1)−1=−3

f(2)=25−3(2)−1=25f(2) = 2^5 - 3(2) - 1 = 25f(2)=25−3(2)−1=25

我们可以看到 f(1)f(1)f(1) 是负数，而 f(2)f(2)f(2) 是正数，即 f(1)<0<f(2)f(1) < 0 < f(2)f(1)<0<f(2)。

根据介值定理，如果一个连续函数在一个区间的两个端点处取不同的符号，那么在这个区间内一定存在至少一个实根。在我们的情况下，由于 f(1)<0<f(2)f(1) < 0 < f(2)f(1)<0<f(2)，因此方程 x5−3x−1=0x^5 - 3x - 1 = 0x5−3x−1=0 在区间 (1,2)(1, 2)(1,2) 内至少有一个实根。

因此，我们成功证明了在区间 (1,2)(1, 2)(1,2) 内方程 x5−3x−1=0x^5 - 3x - 1 = 0x5−3x−1=0 至少有一个实根。
