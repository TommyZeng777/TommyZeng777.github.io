    # 3. 部署到 Github 服务器

    随后输入

     hexo d
    1
    1

    上传部署网页到 github 上， 这里报错概率挺高

    具体解决问题的几个方法


    方法一:

    重复 hexo d 重复尝试， 玄学问题， 大部分情况和网络有关


    方法二:

    干掉根目录下 .deploy_git 文件， git bash 中执行以下命令

    git config --global core.autocrlf false

依次执行

hexo clean    //清除缓存
hexo g        //重新生成相应 deploy 文件
hexo d        //上传服务器并部署新文件
